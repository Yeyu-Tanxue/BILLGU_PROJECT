# BLDC Motor Control / 无刷直流电机闭环控制

This folder contains the public curated files from the BLDC motor control coursework project.

## Highlights

- STM32F10x firmware in C with Keil project files
- Encoder, ADC, OLED, timer, key, LED, and support drivers
- PCB, schematic, and experiment report PDFs in `docs/`
- Focus areas: closed-loop control, motor feedback, PID/FOC-related modules, and embedded hardware debugging

## Notes

The original archive also included IDE caches and generated outputs. Those files are intentionally excluded from this public repository.
