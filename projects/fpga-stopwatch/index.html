FPGA / Verilog / EGo1

FPGA 多功能数字秒表

A Vivado project implementing a stopwatch and countdown timer with debounced buttons, lap memory, LED status output, and six-digit seven-segment display scanning.

100MHzClock divided into 10ms and 1ms ticks
5 KeysStart, preset, pause, mode, lap review
6 DigitsDynamic seven-segment display output
top_stopwatch clk_divider debounce x5 stopwatch_ctrl seg_driver Modes: up-count countdown pause/resume lap review

Design focus

Readable digital logic with hardware constraints.

项目保留了核心 Verilog 源码、EGo1 引脚约束和 Vivado 工程入口。展示重点是有限状态机、 BCD 计数、按键消抖、扫描显示和板级引脚映射。

Vivado generated run directories are intentionally excluded so the repository stays focused on source-level review.
